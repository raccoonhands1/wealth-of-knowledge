START [!concept]
A method defined by computer architecture to perform simultaneous operations on the same piece of data. For example, if you want to add a series of numbers 1, 2, 3, 4, ... 99999, SIMD would perform the operation like [1+2], [3+4], [5+6] ... and the amount of pairs able to be computed simultaneously would depend on the real hardware limitations
END

> [!note]
> not to be confused with [[SIMT]]
