START [!concept]
a general part of backend, used to define params inside of queries ie: ```localhost:3000/my_cat?age=20&breed=idk```
END

> [!faq]
> why use query params?
> for conventions for get requests, sharable URLs (easily replicated without postman or something else) and to specify object ownership
