>[!definition]
> a constant whose value is determined at runtime. ie: const function parameters, const objects whose initializers are non const or other runtime consts

> [!important]
> these definitions are not strict! ie: the compiler can optimize this stuff out as per the [[as-if rule (compilers)]]. You can just avoid these terms since they're not very useful

> [!see]
> [[compile time constant]]
