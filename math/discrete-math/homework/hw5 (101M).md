## 1
prove the handshake lemma by induction on the number of edges. that is,
$$
\sum_{v \in V} d(v) = 2|E|
$$
for any undirected graph $G = (V,E)$

base case:
where |E| = 0, every vertex has degree 0 so the sum is 0 = 2(0) = 2|E|
IH:
suppose the claim holds on all undirected graphs with A edges with A = |E| - 1 >= 0
IS:
then let G=(V,E) be a graph with |E| = A + 1 >= 1
we want to prove
$$
\sum_{v \in V} d(v) = 2(A+1)
$$
since |E| = A+1 >= 1, pick any edge $e \in E$ and write $e = \{u,w\}$ where u=w is allowed if e is a loop
let $G^\prime = (V, E \setminus \{e\})$ which has the same vertex set and A edges
let $d^\prime(v)$ denote the degree of v in $G^\prime$
by the IH applied to $G^\prime$:
$$
\sum_{v \in V} d^\prime(v) = 2A
$$
now compare d(v) to $d^\prime(v)$. removing e affects degrees as follows:
if e is not a loop (u != w), then $d(u) = d^\prime(u) + 1$ and $d(w) = d^\prime(w) + 1$, and $d(v) = d^\prime(v)$ for all other v
if e is a loop (u = w), then by convention a loop contributes 2 to d(u), so $d(u) = d^\prime(u) + 2$ and $d(v) = d^\prime(v)$ for all other v
in both cases, summing over all vertices gives
$$
\sum_{v \in V} d(v) = \sum_{v \in V} d^\prime(v) + 2 = 2A + 2 = 2(A+1) = 2|E|
$$
thus the claim holds by induction
## 2

prove that if a directed graph $G = (V,E)$ contains a closed walk, then it contains a cycle

proof:
suppose G contains a closed walk. among all closed walks in G, choose one of minimum length: $W= v_{0}, v_{1}, \dots, v_{k}$ where $v_{0} = v_{k}$ and $k \geq 1$

a minimum exists since the set of lengths of closed walks in G is a subset of $\mathbb{N}$ which is well-ordered

suppose, by contradiction, that some vertex repeats amidst $v_{0}, \dots, v_{k-1}$, that is $v_{i} = v_{j}$ with $0 \leq i < j \leq k-1$

then $v_{i}, v_{i+1}, \dots, v_{j}$ is a closed walk of length $j - i < k$, contradicting the minimality of W
thus $v_{0}, \dots, v_{k-1}$ are all distinct, meaning W is a cycle by definition
## 3
prove that a DAG must have a source vertex

proof:
pick any vertex in a DAG, $v_{0} \in V$. construct a sequence of vertices $v_{0}, v_{1}, v_{2}, \dots$  such that at each step, if $v_{i}$ has an in-degree of 0, stop. otherwise pick any predecessor vertex $v_{i+1}$ where $(v_{i+1}, v_{i}) \in E$ 

suppose for contradiction that this process never terminates. then we get an infinite sequence $v_{0}, v_{1}, v_{2}, \dots$ with $(v_{i+1}, v_{i}) \in E$ for every i (not a DAG)

since V is finite, by pigeonhole there exist indices $i < j$ with $v_{i} = v_{j}$

then $v_{j} \to v_{j-1} \to \dots \to v_{i+1} \to v_{i} = v_{j}$ is a closed walk in G, which by extension means it contains a cycle

this contradicts G being a DAG, so the process must terminate

the process only stops at a vertex with in-degree 0, which is a source of a DAG by definition
