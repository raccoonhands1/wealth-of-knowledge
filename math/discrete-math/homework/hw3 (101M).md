## 1
use the formal definition of asymptotic notation to prove the following
### a) 
$$
5n^3-10n^2+100=O(n^3)
$$
$$

\begin{align}
\text{find constants c > 0 and $n_{0}$ > 0 such that for all n > $n_{0}$ } c*n^3 \geq 5n^3-10n^2+100 \\


O(n^3) = c *n^3 &\geq 5n^3-10n^2+100 \\
c &\geq 5 - \frac{10}{n} + \frac{100}{n^3}
\end{align}
$$
we see that as n grows to infinity, c is best described as being 5
